## Que se hizo en este chat

Se resolvio un bug de UX que no estaba en los issues del repo.
El sintoma era: navegar a /transacciones dejaba la pantalla en blanco,
y volver al dashboard con el Back de Chrome tambien lo dejaba en blanco
hasta hacer F5.

Adicionalmente se reorganizo el sidebar moviendo Backup dentro de Settings.

Se crearon scripts de mantenimiento para cerrar chats ordenadamente.

## Causa raiz del bug

La API /api/v1/catalogos/categorias?flat=true devuelve { "items": [...] }
pero el componente CategoriaTreeSelect asumia un array directo y llamaba
.find() sobre el objeto, lanzando TypeError: categorias.find is not a function.
Sin Error Boundary, React desmontaba el arbol completo dejando todo en blanco.
